# Tauri Focus — Контекст проекта

## Обзор проекта

**Tauri Focus** — это простое десктопное приложение для быстрых заметок, написанное на **Tauri v2** (Rust бэкенд + React/TypeScript фронтенд). Приложение работает в системном трее и появляется по горячей клавише для быстрого ввода текста, который сохраняется в файл в домашней директории пользователя.

### Ключевые особенности

- **Быстрый доступ**: Окно появляется по `Ctrl + Shift + Q`
- **Работа в трее**: Приложение всегда в трее, не занимает место на панели задач
- **Автосохранение**: При потере фокуса или нажатии `Esc` окно скрывается, текст сохраняется
- **Сохранение в файл**: Заметки сохраняются в `~/tfocus_tasks.txt` с временной меткой
- **Многоязычность**: Установщик поддерживает английский и русский языки

## Сборка и запуск

### Требования (Windows)

1. **Microsoft C++ Build Tools**
2. **Node.js** (v22+)
3. **Yarn** (`npm install -g yarn`)
4. **WebView2** (встроен в Windows 10+)
5. **Rust** (`rustup default stable-msvc`)
6. **VBScript** (для MSI установщиков)

### Команды

```bash
# Запуск в режиме разработки (hot reload)
yarn tauri dev

# Сборка релиза (создаёт .exe и .msi установщики)
yarn tauri build

# Отдельно фронтенд
yarn dev          # Vite dev server
yarn build        # Сборка фронтенда
yarn preview      # Предпросмотр сборки
```

|---------|----------|

| `toggle_app` | Переключить видимость окна |

## Логика работы

### Поток ввода заметки

1. Пользователь нажимает `Ctrl + Shift + Q` → окно появляется
2. Вводит текст → нажимает `Enter`
3. Текст форматируется: `DD-MM-YYYY HH:MM | <текст>`
4. Записывается в начало файла `~/tfocus_tasks.txt`
5. Окно скрывается в трей

### Скрытие окна

- **`Esc`**: Скрыть окно
- **Потеря фокуса** (клик вне окна): Автоматическое скрытие
- **Трей**: Меню "Показать окно" / "Quit"

## Стиль кода

### Rust
- Модульная структура: основная логика в `lib.rs`, утилиты в отдельных модулях
- Использование `#[tauri::command]` для экспорта функций
- Обработка ошибок через `Result<T, E>`
- Dev-логирование через `#[cfg(dev)] println!`

### TypeScript/React
- Функциональные компоненты с хуками
- Строгий режим TypeScript (`strict: true`)
- Tailwind CSS для стилизации
- Асинхронная работа с Tauri API через `invoke()`
